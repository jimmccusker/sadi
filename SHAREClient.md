# Quick start #

Download the [SHARE command-line client](http://code.google.com/p/sadi/downloads/detail?name=SHARE-client-0.1.jar).

Execute as follows:
```
$ java -Xmx1024m -jar SHARE-client-0.1.jar </path/to/query.sparql
```
